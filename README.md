# VLM Paper Review KO

VLM, LLM, VLA 분야의 주요 논문을 한국어로 리뷰하고 정리하는 저장소입니다.

📖 **문서 사이트**: [vlm-paper-review-ko.readthedocs.io](https://vlm-paper-review-ko.readthedocs.io/en/latest/)

## 카테고리

| 분류 | 설명 |
|------|------|
| **VLM** | Vision-Language Model |
| **LLM** | Large Language Model |
| **VLA** | Vision-Language-Action |

## 리뷰 구성

각 논문 리뷰는 아래 항목을 중심으로 정리합니다.

- **핵심 아이디어** — 논문이 해결하려는 문제와 제안 방법의 직관
- **방법론** — 모델 구조, 학습 방식, 주요 설계 선택
- **실험 결과** — 주요 벤치마크 성능 및 ablation
- **한계점** — 논문이 인정하거나 드러난 제약
- **인사이트** — 후속 연구 연결고리 및 개인 코멘트
